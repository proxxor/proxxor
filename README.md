# Proxxor Website

This is the source code for the Proxxor website.

## Deployment

To deploy this website to GitHub Pages:

1.  Create a new repository on GitHub (e.g., `proxxor-website`).
2.  Push this code to the repository:

    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/proxxor-website.git
    git branch -M main
    git push -u origin main
    ```

3.  Go to the repository settings on GitHub.
4.  Navigate to **Pages**.
5.  Under **Source**, select **Deploy from a branch**.
6.  Select the `main` branch and `/ (root)` folder.
7.  Click **Save**.

Your site will be live at `https://YOUR_USERNAME.github.io/proxxor-website/`.
